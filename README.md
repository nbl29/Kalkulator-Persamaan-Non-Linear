# Kalkulator Persamaan Non-Linear
Tampilan terminal yang diperindah menggunakan library `rich`.

## Instalasi Dependensi
```
pip install rich
```

## Cara Menjalankan
```
python main.py
```

## Fitur Tampilan Baru
- Banner ASCII art berwarna di menu utama
- Panel berwarna dengan border rounded
- Tabel iterasi dengan highlight otomatis
- Baris pergantian tanda di-highlight di Metode Tabel
- Warna f(x) berubah hijau saat mendekati nol
- Panel hasil akhir dengan warna & ikon
- Input prompt berwarna dan berikon
